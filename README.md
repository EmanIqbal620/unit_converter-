# Unit Converter ⚖️

> A Python + Streamlit app that converts between units across length, weight, temperature, speed, and more — deployed and publicly accessible.

[![Live App](https://img.shields.io/badge/Live_App-FF4B4B?style=for-the-badge&logo=streamlit&logoColor=white)](https://your-live-link.streamlit.app)

## Conversions Supported

| Category | Units |
|---|---|
| Length | km, miles, meters, feet, inches |
| Weight | kg, pounds, grams, ounces |
| Temperature | Celsius, Fahrenheit, Kelvin |
| Speed | km/h, mph, m/s |

## Tech Stack
`Python` `Streamlit`

## Run Locally
```bash
git clone https://github.com/EmanIqbal620/unit_converter-
cd unit_converter-
pip install -r requirements.txt
streamlit run app.py
```
